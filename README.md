# Traffic Light Controller using Verilog HDL

## Overview
Designed and simulated a Traffic Light Controller using Finite State Machine (FSM) in Verilog HDL.

## Features
- North-South / East-West control
- Red, Yellow, Green transitions
- Safe timing sequence
- Reset support

## Tools Used
- Verilog HDL
- ModelSim / Vivado / Xilinx ISE

## Files
- traffic_light_controller.v
- traffic_light_tb.v

## Applications
Used in smart traffic junction systems and digital control systems.
